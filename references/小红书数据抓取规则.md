# 小红书数据抓取规则

> 本文件是 SKILL.md §2.7 小红书决策树的**技术深度参考**。
> 决策树主逻辑见 SKILL.md，本文件提供安装引导和技术细节。

## 核心结论

小红书有**强登录态依赖**，唯一能拿到完整数据（正文+图片+互动+评论）的方案是 **xiaohongshu MCP**。

### 方案完整性对比

| 方案 | 正文 | 图片 | 互动数据 | 评论 | 完整性 | 前置条件 |
|------|:----:|:----:|:-------:|:----:|:------:|---------|
| **xiaohongshu MCP** | ✅ | ✅ | ✅ | ✅ | **>95%** | 下载二进制+扫码登录+启动服务 |
| web_fetch + xsec_token | ✅ | ❌ | ❌ | ❌ | **~50%** | 链接含 xsec_token |
| web_fetch 裸链 | ⚠️ | ❌ | ❌ | ❌ | **<30%** | 无（极易失败） |

> 已安装的「小红书 Skill」（`~/.workbuddy/skills/小红书/`）是 MCP 的 shell 脚本封装层，底层依赖相同，需要 MCP 服务运行。

## MCP 安装引导（AI 引导用户执行，约 3 分钟）

```bash
# 1. 检测系统架构
uname -m  # arm64 = Apple Silicon, x86_64 = Intel

# 2. 下载对应二进制
# https://github.com/xpzouying/xiaohongshu-mcp/releases
# macOS Apple Silicon:
curl -L -o xiaohongshu-mcp https://github.com/xpzouying/xiaohongshu-mcp/releases/latest/download/xiaohongshu-mcp-darwin-arm64
curl -L -o xiaohongshu-login https://github.com/xpzouying/xiaohongshu-mcp/releases/latest/download/xiaohongshu-login-darwin-arm64
chmod +x xiaohongshu-mcp xiaohongshu-login

# 3. 首次登录（扫码，只需一次）
./xiaohongshu-login
# → 打开浏览器显示二维码 → 用小红书 App 扫码

# 4. 启动服务
./xiaohongshu-mcp
# 服务运行在 http://localhost:18060

# 5. 验证
curl -s http://localhost:18060/mcp | head -1

# ⚠️ 登录后不要在其他浏览器登录同一小红书账号（会使 session 失效）
# ⚠️ Cookie 有效期约 30 天，过期需重新扫码
```

## 失败处理

| 症状 | 原因 | 处理 |
|------|------|------|
| 403 / 验证页面 | 登录态失效 | 重新扫码登录 |
| 空内容返回 | 裸链接无 token | 要求用户提供带 xsec_token 的完整链接 |
| MCP 进程不存在 | 服务未启动 | 运行 `./xiaohongshu-mcp` |
| 频率限制 | 请求太快 | 增加间隔至 3-5 秒 |

## 最佳实践

1. **用户教育**：告知如何获取带 token 链接（浏览器登录后复制地址栏完整 URL）
2. **降级预告**：明确告知 web_fetch 只能拿到文字，缺少图片和互动数据
3. **批量采集**：使用 MCP/Skill 的分页接口，避免 web_fetch 逐条抓取

### 详细分析

**🥇 web_fetch + token — 开源第一选择**
- **零依赖**：任何 AI 平台都有内置 web_fetch，不需要安装任何东西
- **唯一前提**：用户提供带 `xsec_token` 的完整 URL（浏览器登录后复制地址栏）
- **局限**：只有文字，无图片/互动数据
- **结论**：开源用户的"保底方案"，必须写在第一位

**🥈 小红书 Skill（内置）— 被低估的方案**
- 已安装的平台通常内置小红书 Skill，能搜索/详情/评论
- 底层依赖 MCP 服务，但封装更友好
- 适合已有 MCP 环境的用户

**🥉 xiaohongshu MCP — 数据最全但有门槛**
- 需要下载二进制 + 扫码登录（需中国手机号小红书账号）
- 一次配置长期可用，数据最完整
- 适合"进阶用户可选推荐"，不适合作为默认首选
